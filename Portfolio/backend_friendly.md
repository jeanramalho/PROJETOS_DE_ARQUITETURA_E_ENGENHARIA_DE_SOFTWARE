
# Friendly Backend

 -  A proposta desse projeto é criar um frontend que será uma forma amigavel e salvar informações que vão alimentar o portfólio principal.

 ## Arquitetura

 ### Sessões
  
  - Deverá ser possível cadastrar no banco de dados as sessões do portfólio, como Home, Sobre, Projeto, Contato e etc... Também será possível criar novas sessões.
