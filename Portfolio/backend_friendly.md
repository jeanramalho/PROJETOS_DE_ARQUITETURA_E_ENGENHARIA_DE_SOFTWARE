
# Friendly Backend

 -  A proposta desse projeto é criar um frontend que será uma forma amigavel e salvar informações que vão alimentar o portfólio principal.

 ## Arquitetura

 ### Sessões
  
  - Deverá ser possível cadastrar no banco de dados as sessões do portfólio, como Home, Sobre, Projeto, Contato e etc... Também será possível criar novas sessões.

 ### Tipo de exibição 

  - Ao Cadastrar uma sessão o usuário também poderá escolher o tipo de exibição para salvar no banco de dados, para que o projeto de portfolio principal leia e exiba conforme o escolhido. Os tipos de exibição serão tipo, vitrine, carrossel, home, sessão e etc...
